start/3

start(A, B, Base) -> ok

  • A = string()
  • B = string()
  • Base = integer()

Adds two numbers A and B, specified as strings, in the given Base, by spawning child processes to do the job. It spawns 3 child processes by default.

Example

  1> A = "123".
  "123"
  2> B = "654".
  "654"
  3> add:start(A, B, 10).
  "777"

start/4

start(A, B, Base, Options) -> ok

  • A = integer()
  • B = integer()
  • Base = integer()
  • Option = atom() | tuple()
  • Options = [Option]

Adds two numbers A and B, specified as strings, in the given Base, by spawning child processes to do the job. It accepts one option: {'N', N}, where N is the amount of child processes to spawn to go the job.

Example

  1> add:start("12345678", "87654321", 10, [{'N', 5}]).
  "99999999"
